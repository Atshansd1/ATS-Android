# ATS - Attendance Tracking System (Android)

Modern Android attendance tracking application for IT-ADC company.

## Features

- ✅ Check-in/Check-out with GPS location
- ✅ Real-time map showing active employees
- ✅ Attendance history with detailed records
- ✅ Dashboard with statistics and activity feed
- ✅ Multi-language support (English/Arabic)
- ✅ Role-based access (Admin, Supervisor, Employee)
- ✅ Reports generation
- ✅ Employee management

## Download

Download the latest APK from [Releases](https://github.com/mohanadsd/ATS-Android/releases/latest)

## Installation

1. Download the APK file from the releases page
2. Enable "Install from Unknown Sources" in your Android settings
3. Install the APK
4. Sign in with your employee credentials

## Requirements

- Android 8.0 (API 26) or higher
- Internet connection
- Location permissions
